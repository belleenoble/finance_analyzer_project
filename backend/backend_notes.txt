-----> HEALTH CHECK ROUTE <-----

This is another GET endpoint
It is commonly used to confirm the backend is alive
@app.get("/health")
def health_check():
    """
    This function runs when someone visits:
    http://127.0.0.1:8000/health
    """

    return {"status": "ok"}
-> -------------------- <-

----> FILE UPLOAD ROUTE <----

@app.post("/upload")
async def upload_file(file: UploadFile = File(...)):

    - This endpoint recieves a file from the frontend.

    Breakdown of the parameter:
        - file = variable name we will use inside the function
        - UploadFile = special FastAPI type for uploaded files
        - File(...) = tells FastAPI this input must come from a file upload (...) MEANS ITS required
    
    return {
        "filename": file.filename, -> temp
        "message": "File recieved successfully"
    }
-> -------------------- <-
